# MVP 요구사항 정의서

## 프로젝트 개요
인스타그램 계정의 데이터를 분석하여 인사이트를 제공하고, 콘텐츠 개선 방향을 추천하는 서비스

## MVP 핵심 기능 (P0) - "이것만 되면 출시 가능"
**인스타그램 계정 연동 및 기본 성과 대시보드**
- 사용자가 자신의 인스타그램 비즈니스 계정을 연동하고, 최근 30일간의 핵심 지표를 한눈에 볼 수 있는 대시보드

## 상세 요구사항

### 1. IG Graph API 연동 (P0)
**기능 요구사항:**
- OAuth 2.0 기반 인스타그램 비즈니스 계정 인증
- 액세스 토큰 안전한 저장 및 자동 갱신
- 기본 계정 정보 조회 (프로필, 팔로워 수)
- 최근 미디어(게시물) 목록 조회
- 각 게시물의 인사이트 데이터 수집

**API 제약사항:**
- Rate Limit: 200 calls/hour/user
- 비즈니스 계정만 인사이트 접근 가능
- 데이터는 최대 2년까지만 조회 가능
- 스토리 인사이트는 24시간 이내만 가능

### 2. 대시보드 (P0)
**핵심 지표:**
- 총 팔로워 수 및 증감률
- 평균 도달 수 (최근 30일)
- 평균 참여율 (좋아요 + 댓글 / 도달 수)
- 최고 성과 게시물 Top 5

**UI 요구사항:**
- 모바일 우선 반응형 디자인
- 실시간 데이터 로딩 표시
- 그래프를 통한 시각화 (선 그래프, 막대 차트)

### 3. 기본 추천 기능 (P1)
**분석 기반 추천:**
- 최적 게시 시간대 (팔로워 활동 시간 기반)
- 성과 좋은 콘텐츠 유형 분석
- 해시태그 성과 분석 및 추천

### 4. 데이터 수집 자동화 (P1)
- 일 1회 자동 데이터 동기화
- 변경사항 알림 (급격한 지표 변화)
- 데이터 히스토리 저장

### 5. 추가 기능 (P2)
- 경쟁 계정 비교 분석
- AI 기반 캡션 추천
- 콘텐츠 캘린더
- 팀 협업 기능

## 기술적 제약사항
- Supabase 무료 티어 한계 고려 (500MB 스토리지)
- Vercel 무료 배포 제한 (함수 실행 시간 10초)
- API Rate Limit 대응 전략 필요

## MVP 완료 기준
1. 사용자가 인스타그램 계정을 연동할 수 있다
2. 대시보드에서 핵심 지표 4개를 확인할 수 있다
3. 데이터가 매일 자동으로 업데이트된다
4. 3일 이내 구현 가능한 범위
