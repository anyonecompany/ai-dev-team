## 대시보드 현황 (클로드 지침 및 스킬 등 시각화/)

### 완료된 기능 (7개 탭)
1. **팀 현황** — Notion DB 실시간 조회 (PROJECT + KMONG DB), 필터링(전체/자체/KMONG/완료), 정렬, 접힘 섹션
2. **컨트롤 패널** — claude CLI 직접 실행, SSE 스트림, 멀티 세션, 원클릭 커맨드 (/quick, /auto 등)
3. **모니터** — 실시간 세션 테이블, 파일 변경 추적 (3초 폴링)
4. **마스터 헌장** — CLAUDE.md v3.2.0 파싱, 규칙 9개, 템플릿 3개, 지식 베이스 6개
5. **프로젝트** — 11개 프로젝트 카드 + 핸드오프 + 메모리 뷰어
6. **인테그레이션** — Notion/Slack 상태 + 훅 파이프라인 시각화
7. **에이전트** — 12개 에이전트 카드 + 33개 커맨드 + 파이프라인 다이어그램

### 기술 스택
- React 19 + Vite 6 + dotenv
- 백엔드: vite.config.js 미들웨어 (18개 API 엔드포인트)
- 디자인: ui-ux-pro-max 생성 다크 모드 (Fira Code + Fira Sans)
- Notion API 연동: PROJECT_DB + KMONG_DB 실시간 조회, 폴백으로 정적 JSON

### 파일 구조
- `vite.config.js` — 백엔드 API 18개 (~780줄)
- `src/App.jsx` — 프론트엔드 7탭 (~1400줄)
- `data/team-status.json` — Notion 폴백 데이터
- `design-system/ai-dev-team-dashboard/MASTER.md` — 디자인 시스템
