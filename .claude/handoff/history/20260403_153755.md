## 완료된 작업

### AI 엔진 전환 (Gemini → Claude)
- `api-server/main.py`: google-generativeai → anthropic SDK
- `api-server/requirements.txt`: anthropic>=0.42.0
- 환경변수: GOOGLE_API_KEY → ANTHROPIC_API_KEY (Fly.io secrets 설정 완료)
- 모델: claude-sonnet-4-20250514, max_tokens=8192

### 하이브리드 분석 아키텍처
- **서버**: 전체 선수 점수 계산 (attack_efficiency, productivity_per90 등)
- **Claude**: 텍스트만 생성 (team_summary, insights 4개, position_analysis 4개, next_steps)
- **조립**: 서버에서 Claude 텍스트 + 서버 데이터 = 최종 AnalysisResult
- JSON 파싱 안전장치: `_parse_ai_json()` + `_repair_truncated_json()`

### 멀티 시트 xlsx 지원
- `parseFile.ts`: 모든 시트 파싱 → sheets 배열 반환 (헤더 행 자동 감지)
- `FileUpload.tsx`: sheets 배열을 서버에 전달
- `main.py`: 시트 유형 자동 판별 (stats/schedule/match_detail/squad)
- 부가 데이터 추출: 시즌 성적(승/무/패), 스쿼드(평균 연령, 외국인), 경기 기록
- 한국어 상세 포지션 → GK/DF/MF/FW 매핑 (30개+)
- "-" → 0 변환, 카테고리 구분행("  골키퍼" 등) 자동 스킵

### 포천 시민축구단 세일즈 준비
- `pocheon.json` 데모 데이터 생성 (30명, 50골 43어시)
- `name_mapping.json` 포천 32명 추가 (총 71개 매핑)
- 팀 선택 모달 (경주/포천 카드)
- 팀별 로고 동적 표시 (/logos/gyeongju.png, /logos/pocheon.png)

### Claude 프롬프트 고도화
- 역할: "K리그 1부/2부 10년 경력 퍼포먼스 분석관, AFC Pro 라이센스 감독진 보고"
- 인사이트: 전술적 맥락 + 구체적 수치 + 실행 가능한 권고
- 포지션 분석: 주전-백업 뎁스, 경쟁 구도, 보강 필요성
