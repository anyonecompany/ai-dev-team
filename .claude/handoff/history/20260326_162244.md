## 완료된 작업

### PDF/DOCX B&W 컨설팅 형식
- reportlab PDF: 흑백 3색(#000/#333/#999), 세로선 없는 표, 볼드+밑줄 소제목
- python-docx DOCX: PDF와 동일한 구조/스타일로 전면 재작성
- 표지: 구단 엠블럼(14mm) + "Powered by Adapted AI" 텍스트
- 헤더/푸터: 구단명(좌) + 제목(우), 페이지 번호 우측

### 데이터 — 구단 공식 xlsx 연동
- gyeongju_khnp_2025_stats.xlsx → gyeongju.json 파싱 파이프라인
- 4개 시트 파싱 (시즌 일정, 경기 기록, 선수별 출전, 2025 시즌 스탯)
- 시즌 기록: 28경기 12승 9무 7패, 득점 34, 실점 29
- Transfermarkt 디스클레이머 제거 → "구단 공식 기록" 표기
- 징계 위험 인사이트 추가 (경고 3회+ 선수 4명)

### 웹 WYSIWYG — PDF와 동일한 B&W 스타일
- 다크 대시보드 → 흰 배경 줄글 문서 형태 전환
- 인터랙션 유지: 편집 가능 텍스트, 정렬 가능 테이블, 선수 상세 페이지
- RadarChart/PlayerTable/InsightCards 흑백 전환
- 선수 상세 페이지도 B&W 전환

### 핵심 기능 — 웹 편집 → PDF/DOCX 즉시 다운로드
- DownloadButtons: 항상 Fly.io API 호출 (static 파일 제거)
- 편집된 team_summary/insights가 PDF/DOCX에 그대로 반영
- DOCX도 python-docx 서버사이드 생성 (Fly.io)

### 범용 파일 업로드
- CSV + XLSX + XLS 지원 (SheetJS)
- UTF-8 / EUC-KR 인코딩 자동 감지
- XLSX 다중 시트: 데이터 가장 많은 시트 자동 선택
- /api/analyze-raw: raw 데이터를 Gemini에게 통째로 전달
- 컬럼 매핑 불필요 — Gemini가 의미 자동 파악

### 이름 변환 시스템
- Gemini 동적 변환 + name_mapping.json 캐시
- 영문 로마자 → 한국어 (한국인), 한글 음역 (외국인)
