# QA 실행 + 자동 수정

QA를 실행하고, 실패한 항목을 자동으로 수정한다.

## 절차

1. `/qa` 커맨드와 동일하게 전체 QA 실행
2. 실패 항목에 대해:
   - 린트 에러 → 자동 수정 (ruff format, eslint --fix)
   - 타입 에러 → 코드 분석 후 타입 수정
   - 테스트 실패 → 테스트 코드 또는 소스 코드 수정
   - 빌드 에러 → 에러 메시지 분석 후 수정
3. 수정 후 QA 재실행하여 검증
4. 최대 3회 반복 (무한 루프 방지)
5. 결과를 Slack/Monday에 알림
6. 수정 내역을 커밋: `fix: auto-fix QA failures`
