# /build-fix - 빌드 에러 점진 수정

> 아키텍처 문제라면 `/plan`으로 구조적 해결 먼저.

1. 빌드 실행: `npm run build` / `python3 -m py_compile` / `go build ./...`

2. 에러 파싱:
   - 파일별 그룹핑
   - 심각도순 정렬

3. 각 에러마다:
   - 에러 컨텍스트 (전후 5줄) 표시
   - 이슈 설명
   - 수정 제안 및 적용
   - 빌드 재실행
   - 에러 해결 확인

4. 중단 조건:
   - 수정이 새 에러를 만들 때
   - 같은 에러가 3회 반복
   - 사용자 중단 요청

5. 요약:
   - 수정된 에러 수
   - 남은 에러 수
   - 새로 발생한 에러 수

한 번에 하나씩 안전하게!
