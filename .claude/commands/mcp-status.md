# /mcp-status

현재 활성화된 MCP와 도구 수를 점검합니다.
컨텍스트 낭비 방지를 위해 세션 시작 시 실행을 권장합니다.

## 실행
1. 현재 활성 MCP 목록 출력
2. 활성 도구 총 수 카운트
3. 임계값 체크:
   - ✅ 활성 MCP 10개 이하, 도구 80개 이하 → 정상
   - ⚠️ MCP 10~15개 또는 도구 80~120개 → 경고, 비활성화 권고
   - ❌ MCP 15개 초과 또는 도구 120개 초과 → 즉시 정리 필요
4. 현재 태스크와 무관한 MCP 비활성화 제안

## 권장 활성 MCP (ai-dev-team 기준)
항상 켜둘 것:
- supabase (DB 직접 접근)
- github (코드 관리)
- slack (팀 알림)

태스크별 켤 것:
- notion (보고 작업 시)
- browser/chrome (프론트엔드 디버깅 시)
- 기타 프로젝트별 MCP
